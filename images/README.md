# images 폴더 — 멤버 짤 사진 넣는 곳

이 폴더에 멤버 사진 파일을 넣으면 사이트에 자동으로 적용됩니다.
**파일이 없어도 사이트는 정상 동작**하며, 그 자리에 멤버 이름이 적힌
깔끔한 플레이스홀더 박스가 대신 보입니다.

---

## 1) 가장 기본: 멤버당 1장씩

아래 이름 그대로 파일을 넣으면 됩니다. (확장자는 `.png` 또는 `.gif`)

| 멤버 | 기본 파일명 |
|------|-------------|
| 송하영 (Hayoung)   | `hayoung.png` |
| 박지원 (Jiwon)     | `jiwon.png` |
| 이채영 (Chaeyoung) | `chaeyoung.png` |
| 이나경 (Nagyung)   | `nagyung.png` |
| 백지헌 (Jiheon)    | `jiheon.png` |

> 권장 사이즈: **정사각형(1:1)**, 가로세로 800px 이상.
> 카드와 저장 이미지 모두 정사각형으로 가운데를 잘라 꽉 차게 보여줘요.

---

## 2) 멤버당 사진 여러 장 넣기 (시드로 자동 선택)

멤버 한 명에게 사진을 여러 장 넣으면, **생년월일 + 오늘 날짜** 시드에 따라
그날 보여줄 사진이 자동으로 하나 정해집니다.
(같은 사람은 같은 날 같은 사진, 날짜가 바뀌면 다른 사진)

**방법:** 파일을 원하는 이름으로 폴더에 넣고,
`../script.js` 맨 위 `MEMBERS` 의 해당 멤버 `images` 배열에 경로를 추가하세요.

예시 — 송하영에게 사진 3장을 넣는 경우:

```js
{
  nameKo: '송하영',
  nameEn: 'Hayoung',
  images: [
    'images/hayoung.png',
    'images/hayoung-2.gif',
    'images/hayoung-3.png',
  ],
},
```

> 배열 순서·파일명은 자유입니다. `.gif` 도 됩니다(저장 이미지에는 첫 프레임이 들어갑니다).
> 배열에 적었지만 실제 파일이 없으면, 그날은 그냥 플레이스홀더가 보여요.

## 3) 헤더 로고 (`logo.png`)

입력 화면 맨 위(제목 "오늘의 최애는?" 바로 위)에 들어가는 로고입니다.

- 이 폴더에 **`logo.png`** 파일을 넣으면 자동으로 표시됩니다.
- 파일이 없으면 그 자리에 🍓✨ 이모지가 대신 보입니다(사이트는 정상 동작).
- 권장: **정사각형**, 배경 투명(PNG) 또는 흰 배경. 화면에는 높이 84px로 표시돼요.
- 크기를 바꾸려면 `../style.css` 의 `.intro-logo img { height: 84px; }` 숫자만 고치면 됩니다.

## 참고
- 개인이 소장/팬용으로 쓰는 이미지인지 저작권을 확인하고 사용하세요.
- 파일명에는 공백·한글보다 영문 소문자 + 하이픈을 권장합니다. (예: `hayoung-2.png`)
